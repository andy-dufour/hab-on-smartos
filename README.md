## Compiling Rust on SmartOS

Dependencies:

Run these install commands:

```
pkgin in git
pkgin install cmake-3.0.1
pkgin install gccmakedep
pkgin install gcc47-4.7.3nb6
pkgin install gcc49
pkgin install gcc49-libs-4.9
pkgin install py27
pkgin install py27-pip
pkgin install gmake
pkgin install curl
```
